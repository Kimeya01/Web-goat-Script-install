# Web-goat-Script-install
ติดตั้ง Web goat เเบบอัดตาโนโลมัด
`git clone https://github.com/Kimeya01/Web-goat-Script-install.git`
ให้สิทธิ์
`chmod 111 install.sh`
รันไฟล์ 
`./install.sh`
เเล้วจะให้เลือก 
1) โลหดเเละรัน
2) รันเลยหากเคยดาวน์โหลดเเล้ว
